# ctf-commandinjection

This app is obviously not secure and not suitable for production environments, etc.

## Setup

```
git clone https://github.com/AppSecExplained/ctf-commandinjection.git
cd ctf-commandinjection
npm install
node server.js
```
Browse to
`http://localhost:3000`
